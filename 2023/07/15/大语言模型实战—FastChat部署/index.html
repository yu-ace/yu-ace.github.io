大语言模型实战—FastChat部署

1 简介

FastChat是一个用于训练、提供服务和评估基于大型语言模型的聊天机器人的开放平台。其核心特点包括：最先进模型（例如 Vicuna）的权重、训练代码和评估代码。带有 WebUI 和与 OpenAI 兼容的 RESTful API 的分布式多模型服务系统。

本篇主要是以ubuntu系统下的模型部署
